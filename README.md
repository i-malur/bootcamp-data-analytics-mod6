# Bootcamp Data Analytics - Módulo 06 Parte I

## Preview

### O que aprendemos?
* O que são banco de dados?
* O que são banco de dados relacionais?
* O que é dado, informação em conhecimento?
* O que são SGDB?
* Arquitetura cliente-servidor
* Pilares da modelagem de dados (entidades, atributos e relacionamentos)
* Cardinalidades
* Tipos de modelos (conceitual, lógico, físico)
* SQL e sublinguagens (DDL, DML, DQL, DCL, DTL)

### Comandos do SQL
* CREATE, ALTER e DROP
* INSERT, UPDATE e DELETE
* SELECT e FROM
* ORDER BY, LIMIT, DISTINCT
* WHERE
* CASE
* Funções de Agregação (COUNT, SUM, AVG...)
* Agrupamento de dados com GROUP BY e HAVING
* Relacionamento entre Tabelas: JOINs: INNER, LEFT, RIGHT, SELECT e FROM + WHERE
* A Ordem de Execução do SQL

# Bootcamp Data Analytics - Módulo 06 Parte II

## Preview

### O que aprendemos?
* Consultas complexas, com subquery relacionada ou não relacionada;
* CTEs (Common Table Expression);
* Window Functions (com janelas [PARTITION BY] e sem janelas);
* ROW_NUMBER(), RANK() e DENSE_RANK();
* PARTITION BY e ORDER BY;
* Funções de navegação [LAG(), LEAD(), FIRTS_VALUE(), LAST_VALUE()];
* ROWS BETWEEN [UNDBOUNDED PRECEDING AND UNBOUNDED FOLLOWING];
* Manipulação de datas [DATE(), JULIANDAY(), STRFTIME(), GROUP BY()]
* Manipulação de Strings e COnversões [(CONCAT ou ||), SUBSTR(), LENGHT(), UPPER() / LOWER(), REPLACE(), TRIM() / LTRIM() / RTRIM(), CAST()]
* Views e Tabelas Temporárias
* Análise churn
* índices SQL e planos de execução [EXPLAIN e QUERY PLAIN]

### Comandos do SQL
* PARTITION BY e ORDER BY (ASC/DESC)
* ROW_NUMBER()
* RANK() e DENSE_RANK()
* LAG(), LEAD(), FIRTS_VALUE(), LAST_VALUE()
* ROWS BETWEEN
* DATE(), JULIANDAY(), STRFTIME()
* CONCAT (||), SUBSTR(), LENGHT(), UPPER(), LOWE(), REPLACE(), TRIM(), LTRIM(), RTRIM(), CAST()
* EXPLAIN E QUERY PLAIN
